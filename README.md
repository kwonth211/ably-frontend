# 에이블리 프론트엔드 엔지니어 채용 과제

## 개발 환경 설정

### 1. node.js 설치

vite를 이용해 프로젝트가 build되기 때문에 14.18.0 이상의 node.js를 설치해 주세요.

**node.js 사이트에서 설치**

- https://nodejs.org/ko/

**nvm으로 node.js 설치**

```
# 14.18.0 버전이 설치되어 있지 않은 경우 설치해 주세요.
$ nvm install

$ nvm use
```

### 2. 의존성 설치

```
# yarn이 설치되어 있지 않은 경우 설치해 주세요.
$ npm install --global yarn

$ yarn install
```

### 3. 로컬 서버 실행

```
$ yarn dev
```

- http://localhost:3000
